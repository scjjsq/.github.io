实验知识要点

实验原理

蒸馏是利用混合物中各组分沸点不同进行分离和提纯的方法。加热使液体沸腾，产生的蒸气经冷凝后收集，从而实现分离。

当混合液受热时，低沸点的组分首先蒸发，蒸气通过冷凝管冷却后变成液体，从而与高沸点组分分离。

实验现象

1. 加热后烧瓶内液体逐渐沸腾，产生大量气泡

2. 温度计显示温度逐渐上升至液体沸点

3. 蒸气进入冷凝管后液化成液体

4. 冷凝液经牛角管流入锥形瓶

5. 锥形瓶中逐渐积累蒸馏产物

实验步骤

1. 安装仪器：按从下到上，从左到右的顺序安装

2. 加入原料：在蒸馏烧瓶中加入待分离混合物和沸石

3. 通冷凝水：打开冷凝水，确保水流方向正确

4. 开始加热：点燃酒精灯，调节火焰大小

5. 收集馏分：当温度稳定后开始收集

6. 结束实验：先撤火源，后关冷凝水

关键操作要点

  • 温度计位置：水银球置于蒸馏烧瓶支管口处
  • 冷凝水方向：下口进冷水，上口出温水
  • 防暴沸措施：必须加入沸石或碎瓷片
  • 仪器连接：确保装置气密性良好
  • 收集要求：收集到容器约1/3处停止

仪器功能说明

蒸馏烧瓶：盛放待蒸馏液体，支管连接冷凝管

冷凝管：将蒸气冷凝为液体，冷水逆流冷却

牛角管：导流冷凝液至接收容器

锥形瓶：收集冷凝后的蒸馏液

温度计：监控蒸气温度，确定馏分沸点

沸石：提供汽化中心，防止暴沸

仪器安装顺序

1. 固定铁架台位置

2. 放置酒精灯和石棉网

3. 安装蒸馏烧瓶并固定

4. 插入温度计

5. 连接冷凝管

6. 安装牛角管和接收瓶

安全注意事项

1. 加热前必须加入沸石，若中途停止，重新加热前需补加

2. 控制加热速度，保持温度稳定

3. 先通冷凝水，后开始加热

4. 实验结束，先停止加热，后关闭冷凝水

5. 酒精灯使用：用火柴点燃，用灯帽盖灭

应急处理

1. 酒精灯失火：用湿布盖灭，不可用水

2. 液体暴沸：立即撤去热源

3. 烫伤处理：立即用冷水冲洗15分钟

4. 化学试剂接触皮肤：立即用大量清水冲洗
